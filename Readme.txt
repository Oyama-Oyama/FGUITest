FairyGUI官网链接：https://www.fairygui.com/product.html
FairyGUI教程: https://www.fairygui.com/docs/guide/
FairyGUI在线示例：https://www.fairygui.com/docs/examples/

打开项目：
使用安装后的FairyGUI打开以下文件 /FGUIProject/FGUIProject.fairy 

建议设置：
1. 在发布设置中预先设置发布路径
2. 文件放在 /FGUIProject/assets/Package1  目录内，可新建文件夹放置，所有文件不可重名、不可使用中文命名
